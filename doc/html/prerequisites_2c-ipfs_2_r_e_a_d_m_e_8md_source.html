4 ## Quick start for users:
5 * **ipfs init** to create an ipfs repository on your machine
6 * **ipfs add MyFile.txt** to add a file to the repository, will return with a hash that can be used to retrieve the file.
7 * **ipfs cat _hash_** to retrieve a file from the repository
