13 ## Introduction
14 
15 EVEN Network — is an open cross-chain platform with EVEN cryptocurrency that allows users to interact with more than one blockchain at a time.
16 
17 **Cross-chain:** Transactions involving more than one blockchain through a single entry point.
18 
19 **Reliability:** A high degree of protection for digital assets due to unique cryptographic solutions.
20 
21 **Control:** Private keys are always under the control of a user.
22 
23 **Scalability:** The network will continue to run quickly even as the number of participants grows, and external services can be integrated into it using the open API.

44 ## Compiling the EVEN Platform Application from source
45 
46 ### On Linux:
47 
48 (Tested on Ubuntu 18.04 x64)
49 
50 1. Install EVEN Platform dependencies
51 
52  - For Debian distributions (Ubuntu)
53 
54  `sudo apt install build-essential cmake libboost-all-dev miniupnpc libunbound-dev graphviz doxygen libunwind8-dev pkg-config libssl-dev libzmq3-dev libsodium-dev libhidapi-dev`
55 
56 2. Install Qt:
57 
58  *Note*: Qt 5.10 is the minimum version required to build. This makes **some** distributions (mostly based on debian, like Ubuntu 16.x or Linux Mint 18.x) obsolete. You can still build the GUI if you install an [official Qt release](https://wiki.qt.io/Install_Qt_5_on_Ubuntu), but this is not officially supported.
59 
60  - For Ubuntu 17.10+
61 
62  `sudo apt install qtbase5-dev qtbase5-private-dev qt5-default qtdeclarative5-dev qml-module-qtquick-controls qml-module-qtquick-controls2 qml-module-qtquick-dialogs qml-module-qtquick-xmllistmodel qml-module-qt-labs-settings qml-module-qt-labs-folderlistmodel qttools5-dev-tools libqt5websockets5-dev qml-module-qtquick-templates2`
63 
64 3. Clone repository and submodules to root of home directory
65 
66  ```
67  git clone https://github.com/evenfound/even-network.git
68  git submodule init
69  git submodule update
70  ```
71 
72 4. Install cmake
73 
74  `sudo apt-get install cmake`
75 
76 5. Build
77 
78  ```
79  cd even-network
80  QT_SELECT=5 ./build.ubuntu.sh
81  ```
82 
83 The executable can be found in the ./bin folder.
84 
85 ### On Windows:
86 
87 The EVEN Platform on Windows is 64 bits only; 32-bit Windows builds are not officially supported anymore.
88 
89 1. Install [MSYS2](https://www.msys2.org/), follow the instructions on that page on how to update system and packages to the latest versions
90 
91 2. Open an 64-bit MSYS2 shell: Use the *MSYS2 MinGW 64-bit* shortcut, or use the `msys2_shell.cmd` batch file with a `-mingw64` parameter
92 
93 3. Install MSYS2 packages for EVEN Platform dependencies; the needed 64-bit packages have `x86_64` in their names
94 
95  ```
96  pacman -S mingw-w64-x86_64-toolchain make mingw-w64-x86_64-cmake mingw-w64-x86_64-boost mingw-w64-x86_64-openssl mingw-w64-x86_64-zeromq mingw-w64-x86_64-libsodium mingw-w64-x86_64-hidapi
97  ```
98 
99  You find more details about those dependencies in the [EVEN Platform documentation](https://github.com/evenfound/even-network/doc). Note that that there is no more need to compile Boost from source; like everything else, you can install it now with a MSYS2 package.
100 
101 4. Install Qt5
102 
103  ```
104  pacman -S mingw-w64-x86_64-qt5
105  ```
106 
107  There is no more need to download some special installer from the Qt website, the standard MSYS2 package for Qt will do in almost all circumstances.
108 
109 5. Install git
110 
111  ```
112  pacman -S git
113  ```
114 
115 6. Clone repository and submodules to root of home directory
116 
117  ```
118  git clone https://github.com/evenfound/even-network.git
119  git submodule init
120  git submodule update
121 
122  ```
123 
124 7. Build
125 
126  ```
127  cd even-network
128  ./build.msys2.sh
129  ```
130 
131 The executable can be found in the `./bin` directory.
